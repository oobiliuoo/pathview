数据模型与 API 设计

SQLite 表结构 + REST API 端点

SQLite 数据模型
-- 路径表
CREATE TABLE paths (
id INTEGER PRIMARY KEY,
name TEXT NOT NULL,
source_file TEXT,
has_orientation BOOLEAN DEFAULT 0,
point_count INTEGER DEFAULT 0,
color TEXT,
created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
-- 路径点表
CREATE TABLE path_points (
id INTEGER PRIMARY KEY,
path_id INTEGER REFERENCES paths(id) ON DELETE CASCADE,
seq INTEGER NOT NULL,
x REAL NOT NULL,
y REAL NOT NULL,
z REAL NOT NULL,
rx REAL,
ry REAL,
rz REAL
);
-- 列映射配置表
CREATE TABLE column_mappings (
id INTEGER PRIMARY KEY,
path_id INTEGER REFERENCES paths(id) ON DELETE CASCADE,
field TEXT NOT NULL,
csv_column TEXT NOT NULL
);
REST API 端点
POST /api/paths/upload 上传CSV，返回列名+预览数据 POST /api/paths 确认映射，创建路径（含点位数据） GET /api/paths 获取所有路径列表 GET /api/paths/:id 获取路径详情+所有点位 DELETE /api/paths/:id 删除路径（级联删除点位和映射） GET /api/paths/:id/points 获取路径点位（支持分页）
