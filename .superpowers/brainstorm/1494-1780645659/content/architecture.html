核心模块

前端
  • 3D 场景 (react-three-fiber)
  • 路径渲染 (线段 + 点位)
  • 姿态坐标轴 (AxesHelper)
  • 回放动画控制器
  • CSV 列映射对话框
  • 多路径管理面板
后端
  • CSV 上传与解析
  • 路径 CRUD API
  • 列映射配置存储
  • SQLite 数据持久化
  • 静态文件托管 (build)
